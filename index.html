Smart Contract — Liberación Automática

contract FastLedgerRelease {
  address exporter = 0xDE48…f291; // Exportador Alemania
  address senae = 0xEC00…0001; // SENAE Ecuador
  uint256 cifValue = 28175; // USD
  bool arancelPagado = false;
  bool documentosVerificados = true;
  bool inspeccionFisicaOk = false;
  function liberarMercancia() public {
    require(arancelPagado && documentosVerificados && inspeccionFisicaOk);
    emit MercanciaLiberada(msg.sender, block.timestamp, cifValue);
  }
}

Tarifas y Tributos al Comercio Exterior

Basado en el Arancel Nacional del Ecuador — Resoluciones COMEX vigentes.

Arancel Ad Valorem

0%–40%
Sobre valor CIF
Vehículos gasolina: 15–35% · Eléctricos: 0% · Híbridos: 5% · Electrónicos: 0% · Textiles: 15% · Alimentos: 20–40% · Maquinaria: 5%

IVA

5% / 15%
Sobre (CIF + Arancel + FODINFA)
General: 15%. Eléctricos/híbridos: 5%. Exonerado: medicamentos, libros, alimentos básicos.

FODINFA

0.5%
Sobre valor CIF
Fondo de Desarrollo para la Infancia. Aplica a casi todas las importaciones. Se suma a la base del IVA.

Salvaguardias

Variable
Medida temporal COMEX
Vehículos +3.000cc: recargo 5%. Textiles: posible salvaguardia adicional. Activadas por resolución COMEX.

Acuerdo UE-CAN

–5%
Certificado EUR.1
Descuento 5% del arancel para mercancías europeas con Certificado EUR.1 o AT.R. Aplica: DE, ES, IT, FR y toda la UE.
